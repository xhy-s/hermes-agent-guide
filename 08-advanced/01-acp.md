# ACP 适配器

## 8.1 概述

**核心文件**: `acp_adapter/` 目录

**功能**: ACP (Agent Communication Protocol) 适配器，用于 VS Code、Zed、JetBrains 等编辑器的集成。

## 8.3 核心组件

| 文件 | 功能 |
|------|------|
| `acp_adapter/entry.py` | 入口点 |
| `acp_adapter/server.py` | ACP 服务器 |
| `acp_adapter/session.py` | 会话管理 |
| `acp_adapter/auth.py` | 认证 |
| `acp_adapter/client.py` | 编辑器客户端 |
