# Cron 调度系统

## 8.1 概述

**核心文件**: `cron/` 目录

**功能**: 定时任务调度，支持 cron 表达式和一次性任务。

---

## 8.2 架构

```
┌─────────────────────────────────────────────────────────────────┐
│                      Cron 调度架构                               │
└─────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────┐
│  hermes cron                                                    │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │              cron/scheduler.py                           │   │
│  │  - 定时任务调度                                           │   │
│  │  - cron 表达式解析                                       │   │
│  │  - 任务执行                                              │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│  cron/jobs.py                                                  │
│  ┌─────────────────────────────────────────────────────────┐   │
│  │  Job 定义                                                │   │
│  │  - CronJob: 周期性任务                                   │   │
│  │  - OnceJob: 一次性任务                                   │   │
│  └─────────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────────┘
```
