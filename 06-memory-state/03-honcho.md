# Honcho 集成

## 6.1 概述

**核心文件**: `honcho_integration/` 目录

**功能**: Honcho AI 的 dialectic 用户建模集成，跨会话学习用户偏好。

---

## 6.2 Honcho 是什么

Honcho 是一个基于对话的用户建模系统，通过 dialectic（辩证）方法学习用户偏好和心智模型。
